Projects

BOOK-STORE

MERN Stack Project

The book store is built on the MERN stack, which stands for MongoDB, Express.js, React, and Node.js. MongoDB is used as the database to store book information, Express.js is used for building the backend server and handling HTTP requests, React is used for creating the interactive user interface, and Node.js is used for running the server-side JavaScript.

Coordinates Finder

Leaflet.js

Find Coordinates is a mini project that utilizes Leaflet.js to help users find latitude and longitude coordinates of a specific location. By simply entering the name of a place, the application displays a map showing the desired location and provides the corresponding latitude and longitude coordinates.

Speech to Text Converter

React project

The speech-to-text converter project, built using React and Vite, offers a seamless and efficient solution for converting spoken words into written text. By utilizing advanced speech recognition technology, users can simply speak into their device's microphone and see their words transcribed in real-time

Temple project

This project aims to showcase some of the famous temples in India and provide users with their precise locations on an interactive map. By leveraging the power of Leaflet.js, a JavaScript library for creating maps, users can explore and discover these iconic religious sites effortlessly.
